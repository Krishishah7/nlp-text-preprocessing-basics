# NLP Text Preprocessing

This project demonstrates the basic text preprocessing steps used in Natural Language Processing (NLP) using Python and the NLTK library.

The goal of this project is to convert raw text into a clean and structured format that can be used for further NLP tasks such as text analysis or machine learning models.

## üìÑ Dataset

The project uses a custom text file (`sample_nlp_text.txt`) containing text related to Artificial Intelligence and Natural Language Processing concepts.

---

## üß† Preprocessing Steps Performed

The following preprocessing steps are implemented:

1. Conversion of text to lowercase  
2. Removal of punctuation  
3. Tokenization of text into words  
4. Removal of stopwords  
5. Stemming using Porter Stemmer  
6. Lemmatization using WordNet Lemmatizer  

Both **stemming and lemmatization** are applied to show how words are normalized in different ways.
